SolarScope

Detecting Rooftops Ready For Solar Energy

🎯 Overview

SolarScope is an advanced AI-powered application that helps identify rooftops ready for solar energy installation. Using state-of-the-art deep learning models, the system performs sequential detection on satellite and aerial imagery. It first identifies buildings, and if detected, proceeds to locate existing solar panels and assess rooftop suitability for solar energy deployment.

✨ Key Features

  • High Accuracy: State-of-the-art deep learning models for both detection tasks
  • Geospatial Support: Full GeoTIFF support with coordinate-aware detection
  • Interactive Visualization: Real-time visual feedback with overlays and bounding boxes

  • Web-Based Interface: Easy-to-use interface accessible from any browser

🔬 Technical Details

Building Detection Process

The DeepLab V3+ model performs pixel-wise classification to identify building structures. The model processes the input image at 256×256 resolution and produces a binary mask highlighting building regions. A threshold of 1% building coverage is used to determine if buildings are present before proceeding to solar panel detection.

Solar Panel Detection Process

When buildings are detected, the Faster R-CNN model analyzes the image using a sliding window approach (512×512 with 256-pixel overlap by default). This ensures comprehensive coverage of the entire image. Detected solar panels are vectorized and exported as GeoJSON with precise coordinate information.

Performance Considerations

The system is optimized for memory efficiency, with automatic GPU detection and fallback to CPU if necessary. Image resizing and batching strategies minimize memory usage while maintaining detection accuracy.

📋 Supported Image Formats

  • PNG/JPG/JPEG: For building detection only (solar panel detection limited)
  • TIFF/GeoTIFF: Full functionality including geospatial solar panel detection (recommended)

Note: For best results and full solar panel detection capabilities, use GeoTIFF format images.
